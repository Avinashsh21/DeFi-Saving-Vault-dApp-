# DeFi Saving Vault

## Overview
A decentralized application (dApp) that allows users to securely deposit and withdraw ETH using a smart contract.

## Problem Statement
Traditional banking systems are centralized and lack transparency.

## Solution
This project uses blockchain technology to create a trustless savings vault.

## Features
- Deposit ETH into vault
- Withdraw ETH securely
- View balance on blockchain
- Immutable transaction records

## Tech Stack
- Solidity
- Ethereum
- Hardhat / Remix
- (Optional) React.js frontend
